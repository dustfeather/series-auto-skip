Auto Skip for Plex & Netflix

Automatically skips intros and credits so you never have to reach for the remote.

Download Source

Features

⏩

Skip Intros

Automatically clicks the "Skip Intro" button the moment it appears on screen.

⏭

Skip Credits

Jumps past end credits so the next episode starts without delay.

🎬

Netflix

Targets Netflix's built-in skip intro and skip recap buttons directly.

▶

Plex

Works with Plex Web App and local servers at localhost:32400.

🛠

Zero Config

Install it and forget about it. No settings, no popups, no setup required.

⚡

Lightweight

No network requests, no analytics, no storage. Runs entirely in your browser.

How it works

  1. Install the extension in Chrome or Firefox
  2. Navigate to Netflix or Plex and start watching a show
  3. When a skip button appears, the extension clicks it automatically
  4. Sit back and enjoy uninterrupted binge-watching

Install

Chrome / Edge / Brave

  1. Download .zip from Releases
  2. Go to chrome://extensions
  3. Enable Developer Mode
  4. Click "Load unpacked" and select the unzipped folder

Firefox

  1. Download .xpi from Releases
  2. Go to about:addons
  3. Click "Install Add-on From File"
  4. Select the .xpi file

Build from source

$ npm install
$ npm run build

Then load dist/ as an unpacked extension in Chrome or temporary add-on in Firefox.
